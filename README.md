## Run Locally

Clone the project

```bash
  https://github.com/geekychandan/Crypto-user-dashboard.git
```

Go to the project directory

```bash
  cd newproject
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  rarn start
```
